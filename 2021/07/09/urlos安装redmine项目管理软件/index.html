urlos安装redmine项目管理软件

发表于2021-07-09|更新于2021-07-31|工具🔧redmine
|阅读量:

一. 安装urlos

操作系统版本（我的）：ubuntu18.04

  1. ssh连接系统，切换到root用户

    sudo -s

  2. 下载脚本并执行安装命令

    curl -O https://www.urlos.com/siu && sh siu

    如果没有curl工具，执行apt-get install -y curl || yum -y install curl安装curl

  3. 选择服务器所在区域

    国内服务器选1，下载缓慢选2，海外服务器选3

  4. 选择数据的存储目录

  5. 下载安装成功后重启

    安装成功后会自动重启服务器，时间较长（1分钟左右）

  6. 访问urlos

    前提条件：开放9966、9967、9968、9969端口（入站规则）

    打开浏览器通过https://IP:9966或http://IP:9968访问urlos

    用户名与密码在安装中展示了，留意安装成功信息

二. 安装redmine

  1. 登录urlos系统后台，在应用市场中搜索’redmine‘，找到后直接点击安装按钮

  2. 填写基本信息后 - 提交

    • 填写服务名称
    • 选择运行节点
    • 选择智能部署

  3. 初始化redmine，访问http://IP:3000

  4. 初始账号：admin，密码：admin
